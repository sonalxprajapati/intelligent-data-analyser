# 🧠 Intelligent Data Analyzer

An intelligent Streamlit-based application that automatically:
- Analyzes dataset structure
- Suggests suitable analysis types
- Performs EDA with visualizations
- Automatically selects features
- Applies Machine Learning (Regression / Classification)
- Explains decisions using checklist + reasoning

---

## 🚀 Features

- 📊 Dataset Analysis
- 🧠 Decision Engine (Explainable)
- 📈 EDA Visualizations
- 🧹 Auto Feature Selection
- 🤖 ML Model Selection
- 📉 Model Evaluation + Graphs
